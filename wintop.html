Monitor de procesos para Windows

WinTop

htop, pero para Windows.

WinTop es un monitor de procesos para Windows escrito en PowerShell. Liviano, rápido y portable. Te permite ver CPU, memoria, procesos y controlar tu sistema desde la terminal.

Características

Diseñado para mirar el sistema desde la terminal

CPU

CPU por núcleo

Barras por núcleo para ver cómo se reparte la carga cuando Windows permite consultar contadores del sistema.

MEM

Memoria física usada

Lectura clara de memoria física con valores útiles para diagnóstico rápido.

PID

Tabla completa

PID, usuario, prioridad, hilos, memoria, CPU y comando en una vista pensada para operar.

DELTA

CPU precisa

CPU calculada por delta entre muestras, no solo por acumulado desde el inicio.

SORT

Ordenamiento

Ordená por CPU, memoria, PID o nombre sin salir de la consola.

FIND

Filtro operativo

Filtrá procesos por PID, usuario o comando para enfocarte rápido en lo que importa.

KILL

Cierre desde teclado

Cerrá procesos por PID directamente desde la vista interactiva cuando lo necesites.

SAFE

Fallbacks robustos

Continúa funcionando cuando Windows restringe WMI/CIM o consola interactiva.

Instalación rápida

Cloná el repositorio y ejecutá el script desde PowerShell.

git clone https://github.com/maniat1k/wintop.git
cd wintop
.\top.ps1
Abrir repositorio

Uso básico

Ejemplos para iniciar WinTop, ajustar refresco, limitar filas o tomar una sola muestra.

top
top -RefreshRate 1 -ProcessLimit 40 -SortBy mem
top -Once

Atajos de teclado

q salir

h ayuda

c ordenar por CPU

m ordenar por memoria

p ordenar por PID

n ordenar por nombre

f filtrar

k cerrar proceso por PID

+ mostrar más filas

- mostrar menos filas

Requisitos

  • Windows 10/11
  • PowerShell 5.1 o PowerShell 7+
